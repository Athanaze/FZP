# FZP
Clipboard paster with fzf used as an interface.
Written in python 3.

# Installation

`cd /usr/bin && wget "https://raw.githubusercontent.com/Athanaze/FZP/main/fzp" && chmod +x fzp`

`pip install pyperclip`
